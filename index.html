MINESWEEPER CRICKET!

  • Select the grid size you want to play on. The number can be any integer from 5 to 12.
  • Select the gamemode you want to play from the dropdown
  • Click on Play Now! to start a new game.
  • On the grid, click on the blocks in order to play and score points!
  • Enjoy the game!
  • You are the batsman! You need to score as many runs as possible while avoiding all the fielders!
  • There will be eleven fielders against you, placed randomly all over the grid.
  • You have to randomly select the blocks to score runs. Its completely luck dependent!
  • The number of runs you score will be written on the block once you click it.
  • One block can get you either 1, 2, 3, 4 or 6 runs. These runs will add up to your total score!
  • Once you select a block with a fielder, the game will end.
Here are the various game modes you can play!
  • In this mode, a certain number of squares will have No Ball or Wides in addition to the runs.
  • If you select a block with a No Ball, then you will score one additional run. Additionally, on the next ball, if you land on a fielder, the game will not end and you will be able to continue.
  • If you select a square with a Wide, then you will score one additional run. Additionally, on the next ball, you will score double the score of the block you select. However, you are not invincible from fielders.
  • In this mode, we have a shortage of players, so we are resorting to box cricket. The classic box cricket rules will apply!
  • The number of fielders will be lesser. However, you will only be able to score dots i.e 0 runs, 1 run or 2 runs.
  • Five dots and you're out!
